### 🍀Stack

- **Client:** Next.js 14 <br/>
- **Server:** Next.js 14 <br/>
- **Data Management:** Sanity <br/>
- **Data Fetching:** SWR <br/>
- **Auth:** NextAuth <br/>
- **CSS:** Tailwind CSS <br/>
- **Deployment:** Vercel<br/><br/>
