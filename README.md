## 🌷Full-stack web application with Next.js🌷

### 🍀Stack

- **Client:** Next.js 14 <br/>
- **Server:** Next.js 14 <br/>
- **Data Management:** Sanity <br/>
- **Data Fetching:** SWR <br/>
- **Auth:** NextAuth <br/>
- **CSS:** Tailwind CSS <br/>
- **Deployment:** <br/><br/>

### 🍀Getting Started

- `npm run dev` (localhost:3000)<br/>
- `sanity dev` (localhost:3333)<br/><br/>
